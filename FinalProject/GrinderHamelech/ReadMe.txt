## GUI.py
GUI.py is a python file built to connect between the MSP430 micro-processor to a PC-side.
The PC-side will control the functions the micro-processor would do with GUI application which includes control buttons.
include the following libaries - 
PySimpleGUI, time, serial, tkinter, mouse, os, threading, binascii
